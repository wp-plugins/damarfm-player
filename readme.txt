== Description ==



Bu Eklenti Sayesinde Wordpress Blogunuzda DamarFm.Com AAC+ Flash Player Kurulmaktadir.





== Installation ==



1. Wordpress Admin panelinde, EKLENTİLER > Yeni EKLE ve  'DamarFm Player' olarak arama yapın.

2. 'Yükle' Tıklayın.

3. Wordpress Admin içinde , Görünüm > Bileşenler, içinde  'DamarFm.Com Player' istediğiniz bölüme sürükleyip, yerleştirin ve kaydedin..
